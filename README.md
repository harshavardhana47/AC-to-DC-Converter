AC to DC Converter (KiCad Design)

Designed and implemented a AC to DC power converter using KiCad, featuring a full-wave bridge rectifier, filtering stage, and output indication.

🔧 Built using 4× 1N4007 diodes for full-wave rectification
⚡ Integrated 1000µF capacitor for ripple smoothing
💡 Added LED indicator for output status
📐 Ensured clean schematic with ERC validation (0 errors)
🛠️ Ready for PCB design and fabrication
